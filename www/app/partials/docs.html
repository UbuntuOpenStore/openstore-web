How to Install

  1. Download the OpenStore
  2. Open the terminal app
  3. Navigate to the downloads folder
  4. Run the command pkcon install-local --allow-untrusted Downloads/openstore.openstore-team_{{data.version}}_armhf.click

How to use OpenStore with snapd

  1. Download and install snapcraft
  2. Edit /etc/environment and add the following: SNAPPY_FORCE_CPI_URL=https://open.uappexplorer.com/api/snaps/
  3. Restart snapd:
    • For Upstart: service snapd restart
    • For Systemd: systemctl restart snapd
  4. Browse the snapps in the OpenStore: snap find
